# Target Users

## メインユーザー
- 直感型のクリエイター
- 書きながら考える企画職・研究職
- 未完成メモを大切にしたい個人

## サブユーザー
- 日記・内省を習慣化したい人
- 通常メモアプリでは物足りない人
- タスク管理ではなく思考整理をしたい人

## 使う場面
- 移動中に思いついた断片の記録
- 会話後の余韻やキーワードの保存
- 執筆・企画前のアイデア収集
- 定期的な見返しと再構成

## ユーザーが抱える悩み
- 書く時点で整理しきれない
- メモが埋もれて再利用できない
- 重要なテーマを見失う
- アイデアが行動に変わらない

## このアプリが解決すること
- 書くハードルを下げる
- 再浮上で見返し導線を作る
- 関連付けで断片を育てる
- 変換アクションで次の一歩を示す

## 解決しないこと
- 厳密なプロジェクト管理の代替
- 完全自動での思考整理
- SNS型の交流体験
- 初期段階での高精度AI生成依存
