# Seed World

## どんな種か
Kizashi Notes の「種」は、完成文ではない小さな断片です。

- ふと思った一言
- まだ形になっていないアイデア
- 何度も気になるテーマ
- 会話で残った言葉
- 心が動いた瞬間や違和感
- 未来の自分への小さなサイン

曖昧さは欠陥ではなく、成長余地として扱います。

## どんな世界を表現しているか
このアプリは「内側の森」を表現します。

- 種: 小さなメモ
- 芽: 少し育った考え
- 木: 関連種がまとまった単位
- 森: 大きなテーマ群
- 星 / 水脈: 繰り返し現れる言葉
- 浮上する光: 今の自分に必要なメモ

世界観は自然・成長・内省を中心にしつつ、実用メモとして成立させます。

## どんな人に届くか
- アイデアが多く整理しきれない人
- メモを見返せなくなる人
- 創作・研究・企画・発信を行う人
- 日記や内省を続けたい人
- 思考や感性を育てたい人

## どう助けるか
1. **書くハードルを下げる**: 完成前提を外し、一言入力を歓迎する
2. **忘れにくくする**: 再浮上で過去の種と再会できる
3. **育てやすくする**: 問い・タスク・記事案・プロジェクト案へ接続する
4. **流れを見える化する**: 継続テーマや関心変化を捉える
5. **創作・仕事に接続する**: 断片を具体的アウトプットに変換する

## その後どんな世界になるか
- 直感を捨てずに済む
- 未完成の考えが失われず育つ
- メモが情報ではなく未来の芽として機能する
- AI は決定者ではなく創造性を照らす補助になる
- ユーザーが自分の内側の森を継続的に歩ける

## 実装可能な機能への接続
- GrowthState（seed / sprout / tree / archived）で成長段階を管理
- resurfacingScore で再浮上候補を提示
- relatedSeedIds で関係性を表現
- Transform Actions で次行動への橋渡しを提供
