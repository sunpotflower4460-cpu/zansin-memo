# プライバシーポリシー（Kizashi Notes）

最終更新日: 2026-06-07

## 1. 取得する情報
- このアプリは、ユーザーが入力したメモ（種）を端末内に保存します。
- アカウント登録・ログイン機能はありません。

## 2. データの保存場所
- データはユーザー端末内のローカルストレージに保存されます。
- クラウド同期はありません。

## 3. 外部送信について
- AI/API への送信は行いません。
- 広告配信のための外部送信は行いません。
- トラッキングは行いません。

## 4. データ削除
- アプリ内の「すべてのデータを削除」から、端末内データを削除できます。

## 5. サポート連絡先
- サポートは GitHub Issues から受け付けます。
