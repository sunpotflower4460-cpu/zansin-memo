# App Store Readiness (MVP Draft)

## 位置づけ
- このバージョンは **未提出のiOS向けMVP** です。
- 目的は「iPhone上で一通り触れて、MVP体験（書く・残る・再会する・育てる）を確認できる状態」にすることです。
- App Store提出作業そのもの（審査提出・配布設定）は今回のPRでは実施しません。

## 今回のMVPで満たしている前提
- ローカル完結（認証なし・課金なし・クラウド同期なし）
- 外部への個人情報送信なし
- AI/API連携なしでも全機能が成立
- iOSファーストのスマホUI（Safe Area、キーボード重なり軽減を考慮）

## App Storeスクリーンショット候補

### Home
- 今日の種が表示されている状態
- 「今日の種 · M月D日」ヘッダー
- キャッチコピー例: 「毎日、種とやさしく再会する」

### Write
- 本文入力中の状態
- キャッチコピー例: 「きれいにまとめなくても大丈夫」

### Seeds
- 複数の種が並んでいる状態（フィルターチップ付き）
- キャッチコピー例: 「書いた種を、いつでも見返せる」

### Seed Detail
- 本文＋変換結果カードが表示された状態
- キャッチコピー例: 「種を育てる候補を、やさしく提案」

### Garden
- 種/芽/木/保管が色分けされて並んでいる状態
- キャッチコピー例: 「あなたの種が、庭になっていく」
